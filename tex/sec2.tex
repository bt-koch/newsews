\thispagestyle{empty}

Klärung folgender Fragen:
\begin{itemize}
	\item Was ist gegeben? $\rightarrow$ "traditional models" without sentiment as input
	\item Was ist gesucht? $\rightarrow$ does sentiment indicator improve performance?
	\item Was ist das Zielkriterium? $\rightarrow$ new statistically significant and relevant independent variable for estimating risk proxies of banks
	\item Welche Randbedingungen gilt es zu berücksichtigen? $\rightarrow$ maybe trade off between improved performance vs complexity/costs of data?
\end{itemize}
